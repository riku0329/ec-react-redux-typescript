# InFaste

アパレル系ECアプリ

## 作成理由

Zaraのアプリで商品を買う機会が多く、自分をUI/UXの整った使いやすいECアプリを作成したかった。

# 使用技術

React-Redux-TypeScript-Firebase-Stripe

# 機能一覧

## ユーザー側
- ユーザ登録機能
- google認証機能
- カート機能 (localStorage)
- お気に入り機能
- 購入履歴機能

## 企業側
- 製品管理機能
- 売上管理機能
- フィルター機能
- セール機能

## 決済
- stripe決済
- トランザクション機能
